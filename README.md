# EchoArena

Voice-based multiplayer quiz game platform.

## Tech Stack

- Next.js 14 (React)
- Supabase (PostgreSQL + Realtime)
- Daily.co (Voice chat)
- Tailwind CSS
- TypeScript

## Setup

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone repository:

```bash
git clone https://github.com/iam-allend/EchoArena.git
cd echoarena
```

2. Install dependencies:

```bash
npm install
```

3. Setup environment variables:

```bash
cp .env.example .env.local
```

Edit `.env.local` with your credentials:

- Supabase URL & Keys
- Daily.co API Key

4. Run development server:

```bash
npm run dev
```

Open http://localhost:3000
